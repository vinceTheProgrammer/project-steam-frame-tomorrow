Project Steam Frame Tomorrow is a game developed using the Godot game engine by anyone willing to contribute. The idea is to facilitate the development of a creatively diverse game that anyone can play day one of the Steam Frame release. While technical decisions that affect how the game functions will be curated for quality and accessibility, creative decisions will almost always be unrestricted with few exceptions.

Anyone is welcome to contribute by creating as many Pull Requests as you'd like.

Rules:
1. No content that is illegal or breaks GitHub TOS
2. No malicious code
3. No AI generated assets or assets you do not own the rights to
4. No AI generated code or code you do not own the rights to

Guidelines:
- PRs that only add to the repo will almost always be accepted so long as it follows the above rules
- PRs that modify existing code or delete existing code will be accepted on a case by case basis
- PRs that overwrite creative decisions will be denied
- PRs that overwrite technical or backend decisions will usually be accepted as long as they improve functionality/accessibility

Note: 
- Maintainers have final discretion on the exact meaning of these rules/guidelines
